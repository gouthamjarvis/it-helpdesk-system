# IT Helpdesk Intranet System

A Flask-based IT helpdesk and ticket management system hosted on Ubuntu Server and accessible across multiple Windows systems over LAN.

## Features
- Ticket creation system
- Admin dashboard
- Role-based workflow
- Complaint assignment
- Ticket status tracking
- CSV export reports
- Intranet deployment
- SQLite database

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS
- Ubuntu Linux
- Git & GitHub

## Deployment
Hosted on Ubuntu Server inside VMware and accessible across local network systems.

## Future Enhancements
- AI-powered ticket suggestions
- Email notifications
- Analytics dashboard
- Department routing
- Technician management
